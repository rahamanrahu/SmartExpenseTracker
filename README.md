# 📖 About

Smart Expense Tracker is a modern full-stack finance management application built using **Spring Boot** following enterprise development practices.

Instead of simply recording expenses, the system intelligently categorizes transactions, monitors budgets, tracks financial goals, generates AI insights, supports receipt scanning, voice-assisted expense entry, and provides strong authentication with two-factor security.

Designed with scalability, maintainability and clean architecture in mind, the application demonstrates real-world backend engineering suitable for portfolio, academic projects and recruiter evaluation.
